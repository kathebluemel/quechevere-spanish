Level 1 • Unit 3 (Professional Model)

Descriptions & Adjectives

Focus: describing people and objects using ser + adjectives, basic agreement (a/o, s), and high-frequency descriptors. Goal: communicate clear descriptions and interpret them in context.

Stage 1: Understand Stage 2: Interact Stage 3: Create Stage 4: Reflect Stage 5: Perform Progress

← Back to Level 1

Unit Goal + Performance Task

Proficiency target: Novice Low → Novice Mid

Student-friendly goal: By the end of this unit, I can describe people and objects using simple Spanish.

Essential Question: How do descriptions reflect what a culture notices and values?

End-of-Unit Performance Task (Preview):

Create a short “Lost & Found” description (spoken or written) for a missing item and a short description of a person. Goal: be understood using clear, simple descriptors (color/size/qualities) and basic agreement.

Success = clear meaning. Accuracy is emerging.

Stage 1 — Understand (Interpretive)

Your job: get the gist and identify key descriptors (color, size, qualities).

Mini Text A

Mi mochila es grande y negra. Mi cuaderno es pequeño y rojo. Mi profesor es muy amable.

Mini Text B

La estudiante es inteligente y trabajadora. Su teléfono es nuevo y blanco. Su casa es pequeña.

Mini Text C (Lost & Found)

Busco una chaqueta azul. Es grande y muy cómoda. También busco una botella verde, pequeña y vieja.

Stage 2 — Interact (Interpersonal)

Your job: ask for and give simple descriptions (3–4 turns).

Core exchange (practice aloud)

A: ¿Cómo es ____?
B: Es ____ y ____. (color / size / quality)
A: ¿Es grande o pequeña?
B: Es ____.
A: Gracias.

High-frequency adjectives (starter set)

  • grande / pequeño(a)
  • nuevo(a) / viejo(a)
  • bonito(a) / feo(a)
  • amable / simpático(a)
  • inteligente / trabajador(a)
  • cómodo(a) / difícil

Stage 3 — Create (Presentational)

Your job: write clear descriptions of a person and an item.

Sentence frames

  • Mi ____ es ____ y ____.
  • La ____ es ____ y ____.
  • Mi amigo(a) es ____ y ____.
  • Busco un(a) ____ _____. Es ____ y ____.

Stage 4 — Reflect (Culture + Growth)

Your job: connect descriptions to cultural perspectives and reflect on strategies.

Cultural Connection (Products • Practices • Perspectives)

  • Product: Clothing styles and color choices vary across regions.
  • Practice: People may describe others differently depending on what is considered polite.
  • Perspective: What we describe first often reflects values (appearance, personality, community).

Stage 5 — Perform (Assessment)

Complete the performance task with clarity and simple language.

Performance Task: “Lost & Found + Person Description”

  • Item description: 3–4 sentences (color, size, quality)
  • Person description: 3–4 sentences (traits/qualities)
  • Use ser + adjectives, basic agreement

Student-friendly rubric

Category Developing On Target Strong
Task completion Missing parts All parts included All parts + clear detail
Comprehensibility Hard to understand Mostly understandable Easy to understand
Language control Errors block meaning Errors don’t block meaning More accurate + clear

Goal: communicate clearly. Accuracy develops over time.

Teacher Alignment & Standards (click to expand)

ACTFL Modes Alignment

  • Interpretive: interpret short descriptions for gist + key details (color/size/qualities).
  • Interpersonal: ask/answer questions to describe an item/person in a short exchange.
  • Presentational: produce short written/spoken descriptions with emerging agreement.

Proficiency Target

  • Novice Low → Novice Mid (simple descriptions with memorized and recombined language; comprehensibility focus).

Can-Do Statements (Teacher-facing)

  • I can identify key details in simple descriptions (who/what, color, size, qualities).
  • I can exchange information to describe an item/person using simple questions and responses.
  • I can present a short description that is understandable with emerging accuracy.
  • I can reflect on cultural perspectives related to description norms and politeness.

Suggested Pacing

  • 5–8 days: Stage 1 (2), Stage 2 (2), Stage 3 (1–2), Stage 4 (1), Stage 5 (1).
